# Premium UI Asset Style Lock (Grow Simulator)

Use this as the canonical style specification for all generated assets.

## Visual Direction
- Premium mobile game UI art
- Cannabis/grow simulator theme
- Bright green primary accents, cyan for water/system, amber for warnings
- Soft controlled glow, no blown highlights
- Clean dark outlines with consistent thickness
- Strong silhouettes and centered composition
- Transparent background for all icons
- No text, letters, numbers, logos, or watermarks baked into art

## Rendering Constraints
- Icon output: PNG RGBA, square, transparent
- Subject centered in frame with enough padding for readability at small sizes
- Keep shape language rounded and polished, mobile-friendly
- Keep lighting direction consistent across all icons

## Avoid
- Generic clip-art look
- Muddy colors
- Inconsistent outline weight
- Busy micro-details that break at small size
- Any in-image text labels
